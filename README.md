# Database Management Systems - Lab Solutions
![MSSQL](https://img.shields.io/badge/MSSQL-CC2927?style=for-the-badge&logo=microsoft-sql-server&logoColor=white)
![TSQL](https://img.shields.io/badge/T--SQL-blue?style=for-the-badge)

Bu depo, Balıkesir Üniversitesi Bilgisayar Mühendisliği 3. sınıf kapsamında hazırladığım veri tabanı uygulama ve sınav çözümlerini içermektedir.

## Proje İçerikleri

### 01-Basic-SQL-Commands
Temel SQL komutları (DDL/DML), agregasyon fonksiyonları ve basit veri manipülasyonu örnekleri.

### 02-Relational-Database-Analysis
Gelişmiş sorgulama teknikleri, karmaşık JOIN yapıları ve `GROUP BY ... HAVING` filtreleme çalışmaları.

### 03-Library-Management-Automation
İlişkisel tablo tasarımı, saklı yordamlar (Procedures), görünümler (Views) ve otomatik stok yönetimi sağlayan tetikleyiciler (Triggers).

### 04-Cargo-Tracking-Full-Project
Uçtan uca kargo takip sistemi veri tabanı modeli. ER Diyagramı tabanlı tasarım, veri güvenliği protokolleri ve ileri seviye T-SQL otomasyonları.

## Teknik Yetkinlikler
* **Veri Tabanı:** Microsoft SQL Server (MSSQL)
* **Dil:** T-SQL
* **Modelleme:** ER Diyagramı Tasarımı ve Veri Tutarlılığı
* **Otomasyon:** Triggers, Stored Procedures, UDF (User Defined Functions)
